# ResumeLift - Beginner Resume Builder

ResumeLift is a free, beginner-friendly web app to create professional resumes with:
- guided input forms
- live resume preview
- three ATS-friendly templates
- one-click PDF export
- responsive desktop/mobile layout

## Tech Stack
- React + Vite + TypeScript
- Tailwind CSS
- React Hook Form
- Zustand
- html2pdf.js

## Run Locally
```bash
npm install
npm run dev
```

## Quality Checks
```bash
npm run lint
npm run build
```

## Deploy to Vercel
1. Push this folder to GitHub.
2. Open Vercel and import the repository.
3. Build settings:
   - Build command: `npm run build`
   - Output directory: `dist`
4. Click deploy.

## 2-3 Minute Demo Script
1. Start with the problem: students struggle with resume structure and formatting.
2. Open ResumeLift and show responsive behavior on smaller and larger screens.
3. Fill a few form fields or click **Load sample**.
4. Show live preview updating instantly.
5. Switch between **Classic / Modern / Minimal** templates.
6. Click **Download PDF** and show the generated file.
7. Close with impact: free, simple, beginner-first, and job-ready output.
